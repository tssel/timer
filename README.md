## Requirements

- **Python 3.10 or newer** – Download it here: https://www.python.org/downloads/
- PyQt6 (installed automatically with the requirements below)

## Installation
Clone the repository:

```bash
git clone https://github.com/tssel/timer
cd timer
```

Install dependencies:

```bash
pip3 install -r requirements.txt
```

Run the application:

```bash
python3 timer.py
```


## How to Use

- **Start/Pause:** Click the play button or click anywhere on the timer.
- **Settings & Options:** **Right-click** anywhere on the app window to open the menu.
